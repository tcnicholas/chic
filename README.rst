Coarse-graining Inorganic and Hybrid Frameworks (CHIC)
======================================================

This is a set of tools for coarse-graining complex frameworks and analysing and visualising the geometric diversity in crystal structure databases using machine learning.

Requirements
------------
* Python_ 3.6 or later
* NumPy_
* Numba_

* Pymatgen_
* NetworkX_

* DScribe_
* skikit-learn_
* UMAP_
